# File synchronization

Slideshow supports automatic file synchronization across the network from the following sources:

- HTTP / HTTPS / FTP server - [more information](file_synchronization_http_ftp.md)
- Google Drive - [more information](file_synchronization_google_drive.md)
- Dropbox - [more information](file_synchronization_dropbox.md)
- WebDAV
- SFTP

The synchronization can be set up and monitored through [web interface](../web_interface.md) → menu `Tools` → `File synchronization`.

!!! note "Stable connection required"
    Stable network / internet connection is required for the file synchronization, otherwise it may time-out or fail often.

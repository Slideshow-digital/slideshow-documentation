# Documentation

This is documentation for the current version of the Slideshow Digital Signage app for Android.

You can also check our [tutorials on YouTube](https://www.youtube.com/channel/UCJFSXCItojY4PS76JNr_pdQ).

## Where to start

- If you don't already have suitable Android hardware, check out our [recommendations](hardware/suggested-hardware.md).
- You might want to check details about the [local web interface](network-access/web-interface.md), which acts as a main interface for managing Slideshow.
- For displaying more than just media files from a single folder, check configuration for [content and playlists](configuration/playback-configuration/content-playlist.md).
- If you want to use Slideshow without network access, you can copy media files from [USB Flash drive](configuration/file-storage.md#usb-flash-drive-as-a-source-for-copying-or-temporary-playback).
- Lastly, setting up [automatic startup](configuration/automatic-startup.md) is often encouraged.

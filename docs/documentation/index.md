# Documentation

This is documentation for the current version of the Slideshow software. It’s being updated regularly, but it doesn’t cover all features of Slideshow in detail yet.

You can also check our [tutorials on YouTube](https://www.youtube.com/channel/UCJFSXCItojY4PS76JNr_pdQ).

## Where to start

- If you don't already have suitable Android hardware, check out our [recommendations](hardware/suggested_hardware.md).
- You might want to check details about the [local web interface](network_access/web_interface.md), which acts as a main interface for managing Slideshow.
- For displaying more than just media files from a single folder, check configuration for [content and playlists](configuration/playback_configuration/content_playlist.md).
- If you want to use Slideshow without network access, you can copy media files from [USB Flash drive](configuration/file_storage.md#usb-flash-drive-as-a-source-for-copying-or-temporary-playback).
- Lastly, setting up [automatic startup](configuration/automatic_startup.md) is often encouraged.
